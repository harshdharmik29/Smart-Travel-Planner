# 🌍 AI-Powered Smart Travel Planner with Real-Time Condition Awareness

An intelligent travel planning system that recommends personalized trips based on weather, traffic, budget, crowd level, and user preferences using AI/ML.

## 🚀 Features
- Personalized destination recommendation
- Weather-aware planning
- Traffic and crowd prediction
- Budget optimization
- Hotel & route suggestions
- Dynamic itinerary generation
- Real-time alerts

## 🧠 AI/ML Models
- Recommendation System
- Clustering (KMeans)
- Route optimization
- Time-series prediction
- Sentiment analysis for reviews

## 🛠️ Tech Stack
- Python
- FastAPI
- Scikit-learn
- OpenWeather API
- Google Maps API
- Pandas
- Streamlit
